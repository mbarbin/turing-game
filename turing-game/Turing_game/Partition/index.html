Module Turing_game.Partition

val create : Condition.t Nonempty_list.t -> t Core.Or_error.t

The condition given as an input must properly define a partition of the set of all possible codes, otherwise the function will return an Error. To properly define a partition, the condition must be such that each code verifies exactly one of the input condition, and also that no condition yields an empty subset.

val counts : t -> int array

Go over all possible codes, and count the number of elements resulting in each of the subset of the partition. It is guaranteed by construction of t that each subset has at least 1 element.
